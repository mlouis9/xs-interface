The following data file defines a complete set for the following states:

  • 2 history points

  • 3 time points

  • 2 fuel branches

  • 2 mod branches

  • 2 cool branches

This file does not use a filter card as all the states are defined here; resulting in a square/complete matrix. The order at which branches are defined is important when values are processed. Here the order follows the fuel, mod, and cool.

The data covers all the macro, micro, kinetics, and meta variables defined in the settings card.

The data values, which are provided using the data set card (and block) can be provided in any order. Please note that macro and micro data here rely on the 2-group energy structure.

# -----------------------------------------------------------------------------
#            INPUT EXAMPLE FOR A SINGLE CROSS SECTION TYPE
# -----------------------------------------------------------------------------


# -----------------------------------------------------------------------------
#                            SETTINGS
# -----------------------------------------------------------------------------


set settings 2 7
macro =  inf_flx, inf_nsf, inf_rabs, inf_sp0
micro =  sig_f, sig_sct, nd
kinetics =  beta, decay
meta =  date
isotopes = 531350, 541350, 922350
nuclides = nd

# -----------------------------------------------------------------------------
#                            STATES
# -----------------------------------------------------------------------------

set branches 3
fuel 900, 1200
mod 650, 700
cool 550, 600


set histories 2
nom 850 550 570
pert 950 500 525

set times nounits
0.0 0.5 1.0
